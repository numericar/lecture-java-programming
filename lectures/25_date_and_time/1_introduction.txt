Date and Time
    ในปัจจุบัน Java มี package ในการใช้จัดการกับเวลา คือ java.time ซึ่งเข้ามาใน Java 8

    Date Representation
        วิธีการแสดงวันที่ใน Java จะจัดเก็บในรูปแบบ long เนื่องจากเป็นแบบมิลลิวินาที
        โดยเริ่มนับตั้งแต่ 1 Jan 1970

    Gregorian Calender
        เป็นรูปแบบปฏิธินทีใช้ได้ทั้งโลก
